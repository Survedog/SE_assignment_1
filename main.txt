서브 모듈들을 통합한 메인 모듈

[사용자 모드로 실행한 경우]
1) load_word.txt의 단어 로딩 함수를 통해 단어를 배열로 불러온다.
2) show_list.txt의 단어 목록 표시 함수를 통해 단어들을 출력한다.
*) 단어 목록에 터치 이벤트 발생 시 터치된 단어의 추가 정보를 표시한다.
*) 검색 버튼에 터치 이벤트 발생 시 단어 검색 함수로 검색을 실시한다.
*) 버튼 터치 이벤트가 발생할 때마다 효과음을 발생시킨다.

[개발자 모드로 실행한 경우]
1) load_word.txt의 단어 로딩 함수를 통해 단어를 배열로 불러온다.
   - 단어를 모두 불러오면 배열의 내용을 word_pool.txt의 내용과 비교하며 문제가 없나 확인한다.
2) show_list.txt의 단어 목록 표시 함수를 통해 단어들을 출력한다.
*) 단어 목록에 터치 이벤트 발생 시 터치된 단어의 추가 정보를 표시한다.
*) 검색 버튼에 터치 이벤트 발생 시 단어 검색 함수로 검색을 실시한다.
   - 배열을 순회하며 각 단어들을 입력하여 올바른 결과가 나오는 지 확인한다.
*) 버튼 터치 이벤트가 발생할 때마다 효과음을 발생시킨다.
